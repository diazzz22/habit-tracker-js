# Habit Tracker

Simple habit tracker built with HTML, CSS, and JavaScript.

## Features

- Add, edit, and delete habits
- Mark habits as completed for today
- Filter habits by all, pending, or completed
- Track current streaks
- Save data with `localStorage`

## Project Structure

- `index.html`: app structure
- `styles.css`: app styles
- `script.js`: app logic

## How to Use

1. Open `index.html` in your browser.
2. Add a habit.
3. Click a habit to mark it as completed today.
4. Use the filters to view different states.

## Current Goal

The project is evolving from a simple list into a more complete habit tracker, with streak tracking and a future monthly progress view.
